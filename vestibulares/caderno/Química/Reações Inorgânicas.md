Síntese ou Adição
- 2 ou mais substâncias formando apenas 1.
- Exemplo: C + O2 -> CO2

Análise ou decomposição
- Uma substância produzindo duas ou mais.
- Tipos de decomposição
	- Fotólise: Decomposição pela luz
	- Pirólise: Decomposição pelo calor
	- Eletrólise: Decomposição pela passagem da corrente elétrica

Simples Troca ou deslocamento
- Quando existe uma substância simples e uma composta, e ocorre a troca: a substância simples troca de lugar com algum elemento da substância composta.
- Ex: AB + C -> AC + B

Dupla troca
- Quando existe duas substâncias compostas e ocorre a troca de um elemento entre as substâncias compostas.
- Ex: AB + CD -> AB + CB


# Simples Troca
Quando existe uma substância simples e uma composta, e ocorre a troca: a substância simples troca de lugar com algum elemento da substância composta.
A + BC -> B + AC

- Toda simples troca é uma oxi-redução
- Condições
	- Semelhante desloca semelhante (Metal desloca metal e Ametal desloca Ametal)
	- A reação só ocorre se a substância simples for mais reativa que a sua semelhante na composta.
- Filas de reatividade
	- Metais: Familia 1A, 2A, Al, Zn, Fe, Pb, H, Cu, Hg, Ag, Pt e Au
		- Facilidade para perder elétrons.
	- Ametais: F, O, N, Cl, Br, I, S, C, P, H (Fonclsbriscph)
		- Facilidade para dar elétrons
- Exemplos:
	- Metal + metal
		- Zn + CuSO4 -> Cu + ZnSO4
			- Zinco(Zn) é mais reativo que Cobre(Cu)
			- Zinco sofreu oxidação - agente redutor
		- Ca + AgNO3 -> Ca(NO3)2 + Ag
			- Cálcio é mais reativo que o metal nobre Cobre
	- Metal + ácido
		- Zn + HCl -> H2 + ZnCl2
			- H é mais reativo que zinco
			- Quando retirado, o H é instável, sendo necessário ligá-lo com outro hidrogênio. 
		- Fe + HCl -> H2 + FeCl2
	- Metal + água
		- Ca + H2O -> H2 + Ca(OH)2
			- O cálcio troca de lugar com H
		- Na + H2O -> H2 + Na(OH)
			- Elementos da família 1A são extremamente reativos, causando explosões quando em contato com a água.
	- Metal + ametal
		- Cl + NaBr -> Br2 + NaCl
			- Ametais sozinhos não são instável, portanto deve ser acompanhados por mais uma molécula do ametal.

# Dupla Troca
Quando existe duas substâncias compostas e ocorre a troca de um elemento entre as substâncias compostas.
- Ex: AB + CD -> AB + CB

- Condições
	- Formação de produtos insolúveis (bases e sais insolúveis)
	- Formação de produto volátil: baixo ponto de ebulição e alta pressão.
	- Formação de produto fraco: libera poucos íons (ácidos e bases)
- Exemplo
	- CaCO3 + H2SO4 -> CaSO4 + H2CO3 (ou H20 + CO2)
		- CaCO3: Calcário / mármore
		- H2SO4: chuva ácida
		- Trocar os cátions e ânions de posição
		- ácido fraco, volátil. 
	- Ba(OH)2 + H2(SO4) -> BaSO4 + 2H2O
		- Reação base + ácido
		- BaSO4 é insolúvel
	- NaCL + AgNO3 -> NaNO3 + AgCl
		- produto insolúvel
	- NaCN + H2SO4 0 -> Na2SO4 + HCN 
		- Volátil e ácido fraco
	- NaHCO3 + HCl -> NaCL + H2CO3 (ou H20 + CO2)
		- HCl: Ácido estomacal
		- NaHCO3: bicarbonato de sódio
